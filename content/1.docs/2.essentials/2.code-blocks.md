Use blocos de código para guardar comandos e exemplos sem perder formatação.

## Comandos

```bash
pnpm install
pnpm dev
```

## Código

```ts
type Habit = {
  id: string
  name: string
  frequency: 'daily' | 'weekly'
}
```

## Boas práticas

- Prefira snippets curtos e com contexto.
- Se o código for grande, linke para a origem (PR, arquivo, issue) e explique o “porquê”.
